# MyStayApp
Use Case 1: Application Entry & Welcome Message

Description:
This use case represents the starting point of the application.
When the user runs the program, the system displays a welcome message and confirms that the system has started successfully.

Flow:
User runs the application.
JVM calls the main() method.
Application prints welcome message.
Application prints system initialized message.
Program ends.

🔹 Flow of Project

Start the application.
Execute main() method.
Display system welcome message.
Confirm system initialization.
End program.

🔹 Project Procedure 

Created a Java class named UseCase1HotelBookingApp.
Defined the main() method as the entry point.
Used System.out.println() to display messages.
Printed:
Welcome message
System initialization message
Compiled and ran the program to verify output.

🔹 Objective

The objective of this project is to create a Hotel Booking Management System that starts properly and displays a welcome message to the user.
It shows how a Java program begins execution using the main() method.
